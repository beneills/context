# Context

Create desktop /contexts/ with custom entry/exit actions

## Installation

    $ gem install context

## Usage

    $ context new test
    $ cd ~/.context/contexts/vpn/
    $ emacs enter.sh exit.sh
    $ context list
    $ context vpn
    $ context -h

Also, check out examples in examples/ directory.
